# 本地视频文件处理功能使用说明

该功能允许用户上传和处理本地视频文件，从视频中提取帧并进行目标检测。

## 使用Web界面上传视频

1. 启动Web应用服务器：
   ```
   python main.py
   ```

2. 打开浏览器访问：http://localhost:6920

3. 在"数据导入与配置"页面的"视频文件处理"部分：
   - 点击"选择文件"按钮选择本地视频文件(.mp4, .avi等格式)
   - 可选填写视频ID（不填则自动生成）
   - 设置抽帧间隔（默认1秒）
   - 设置最大处理帧数（默认20帧）
   - 点击"上传并处理视频"按钮

4. 系统会在后台处理视频，处理完成后可以在"检测结果查询"页面查看结果。

## 使用命令行处理视频

也可以使用命令行工具直接处理本地视频文件：

```
python process_local_video.py [视频文件路径] [选项]
```

### 命令行选项：

- `--video-id ID` - 设置视频ID（不设置则自动生成）
- `--interval 秒数` - 设置抽帧间隔，默认1.0秒
- `--limit 数量` - 设置最大处理帧数，默认20帧
- `--no-save` - 不保存检测结果图像

### 示例：

```
# 基本用法
python process_local_video.py D:/Videos/测试视频.mp4

# 指定视频ID和抽帧设置
python process_local_video.py D:/Videos/测试视频.mp4 --video-id TEST001 --interval 0.5 --limit 30
```

## 注意事项

1. 视频处理是CPU密集型操作，处理大型视频可能需要较长时间。
2. 建议先使用较小的视频文件进行测试。
3. 系统会自动复制检测结果图像到可访问的静态目录。
4. 设置较小的抽帧间隔和较大的帧限制会增加处理时间和存储需求。
5. 处理完成后，可以通过"检测结果查询"页面按视频ID筛选查看结果。

## 支持的视频格式

- MP4 (.mp4)
- AVI (.avi)
- MOV (.mov)
- MKV (.mkv)
- WMV (.wmv)
- FLV (.flv)

## 测试用视频

如果没有合适的测试视频，可以使用以下方法获取：

1. 使用样本视频网站，如 Pexels Videos (https://www.pexels.com/videos/) 下载免费视频
2. 使用手机录制简短的测试视频
3. 从YouTube等平台下载公开视频用于测试（请遵守版权规定）

在进行实际应用部署前，建议使用各种场景下的真实视频进行充分测试。
